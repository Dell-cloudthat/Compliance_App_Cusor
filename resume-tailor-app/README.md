# Resume Tailor App

Standalone Vite + React app for tailoring your resume to a job posting.

## Run locally

```bash
cd /workspace/resume-tailor-app
npm install
npm run dev
```

Then open the URL printed in your terminal (usually `http://localhost:5173`).

## Features

- Upload **resume** and **job requirements** (TXT, PDF, DOCX)
- Auto-extract text from uploads
- Generate an editable **1-page tailored resume**
- Load sample data to test quickly
